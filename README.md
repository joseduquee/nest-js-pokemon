# Run in Development

1. Clone the repository
2. Run
```
yarn install
```
3. Nest CLI must be installed
```
npm i -g @nestjs/cli
```


4. Run the BBDD
```
docker-compose up -d
```

## Stack
* MongoDB
* NestJS
